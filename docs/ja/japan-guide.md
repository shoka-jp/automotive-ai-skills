# 日本のユーザー向けガイド

本リポジトリのスキルは市場を問わず使える設計ですが、元の文書は英語圏(特に米国)の文脈で書かれています。このガイドは、日本で使う際の読み替え・注意点・活用のコツをまとめたものです。

## まず知っておくこと

- **全スキルに日本語版があります。** 各スキルフォルダの `README.ja.md` を開いてください(例: [`skills/mechanic-assistant/README.ja.md`](../../skills/mechanic-assistant/README.ja.md))。
- **AIへの質問は日本語でOKです。** スキルは「聞き方の構造」を提供するものであり、言語は問いません。日本語で質問すれば日本語で、日本市場の文脈で回答されます。
- **車両情報に「日本仕様」と明記すると精度が上がります。** 同名の車でも国内仕様と輸出仕様で装備・エンジンが異なることが多いため、「2015年式 ホンダ フィット(国内仕様、GK3)」のように型式まで書くのが理想です。

## 米国向け記述の読み替え表

| 英語版の記述 | 日本での読み替え |
|---|---|
| miles(マイル) | × 1.6 ≒ km(例: 100,000マイル ≒ 16万km) |
| smog check / emissions inspection | 車検時の排ガス検査 |
| state inspection | 車検(継続検査)+ 法定点検 |
| NHTSA(リコール検索) | 国土交通省の「自動車のリコール・不具合情報」 |
| Carfax / AutoCheck(履歴レポート) | 修復歴の開示、日本自動車査定協会の検定、オークション評価点 |
| Magnuson-Moss法(保証と社外部品) | 日本には直接の同等法なし。メーカー保証の約款に従う |
| ガソリンのオクタン価 87/91/93(AKI) | レギュラー(RON 90前後)/ ハイオク(RON 100)。表記方式が異なる点に注意 |
| Level 1 / Level 2 充電 | 100V / 200V 普通充電 |
| CCS急速充電 | 日本の急速充電は CHAdeMO(チャデモ)が主流 |

## 日本特有のトピック

### 車検・法定点検

米国の多くの州より検査が厳格で、費用も大きな出費です。[車検前セルフチェックリスト](./shaken-checklist.md) を用意しています。ポイント:

- 車検(検査)と24ヶ月点検(予防整備)は別物 — スキルの [Vehicle Maintenance Planner](../../skills/vehicle-maintenance-planner/README.ja.md) は後者の計画に有効です
- カスタマイズは常に「保安基準適合」「車検対応」を軸に検討してください([Vehicle Customization Advisor](../../skills/vehicle-customization-advisor/README.ja.md) に質問する際は「車検に通るか」を必ず添える)

### 軽自動車

軽自動車(660cc、黄色ナンバー)は海外にほぼ存在しないカテゴリのため、AIの学習データが相対的に薄い領域です。質問時のコツ:

- 型式を明記する(例:「スズキ アルト HA36S」「ホンダ N-BOX JF3」)
- 軽特有の事情を前提として伝える:小排気量ターボの負荷、CVTが主流、リアエンジン車(サンバー等)の存在
- オイル量・タイヤサイズなどは普通車の感覚で答えられる場合があるため、**数値は必ず取扱説明書で確認**

### 中古車購入(日本市場)

[Used Car Inspector](../../skills/used-car-inspector/README.ja.md) を使う際の日本向け補足:

- 「修復歴あり/なし」の定義(骨格部位の修理)を確認の軸にする
- オークション評価点、鑑定書、ワンオーナー・記録簿ありの価値を質問に含める
- 走行距離の妥当性は「年間8,000〜10,000km」を日本の平均として伝える
- 雪国の車は下回りの錆(融雪剤)を重点確認 — チェックリストに含めるよう依頼

### EV・充電環境

[EV Assistant](../../skills/ev-assistant/README.ja.md) を使う際:

- 急速充電はCHAdeMO、普通充電は200Vが前提であることを伝える
- 集合住宅の充電設備事情、基礎充電の可否は日本固有の課題として明示する
- 中古リーフの検討では「セグ欠け」(バッテリー容量計)という日本で定着した指標をそのまま使って質問できます
