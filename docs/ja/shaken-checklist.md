# 車検前セルフチェックリスト(AIスキル活用ガイド付き)

車検(継続検査)の前に自分で確認できる項目をまとめたチェックリストです。本リポジトリのAIスキルと組み合わせることで、「車検で何を指摘されそうか」を事前に把握し、不合格や追加整備費用のサプライズを減らすことを目的としています。

> ⚠️ **免責:** 本チェックリストは一般的な情報であり、法定点検(12ヶ月・24ヶ月点検)や指定整備工場・認証工場による点検整備の代わりにはなりません。保安基準の詳細は変更されることがあります。最終的な適合判断は検査機関・整備工場に委ねてください。[安全免責事項](../safety-disclaimer.md)(英語)も参照。

## 車検の基礎知識

- **有効期間:** 自家用乗用車は新車登録から3年、以降2年ごと。軽自動車も同様(検査は軽自動車検査協会)。バイクは251cc以上が車検対象。
- **受け方:** ディーラー/整備工場に依頼、車検専門店、またはユーザー車検(運輸支局・軽自動車検査協会に自分で持ち込み)。
- **費用の構造:** 法定費用(自動車重量税+自賠責保険+検査手数料)はどこで受けても同じ。差が出るのは**整備費用と代行手数料**。
- **車検と法定点検は別物:** 車検は「検査時点で保安基準に適合しているか」の確認であり、24ヶ月点検は予防整備。車検に通った=2年間安心、ではありません。

## セルフチェックリスト

### 灯火類(不合格理由の定番・自分で直せる)

- [ ] ヘッドライト:ロー/ハイとも左右点灯、レンズの黄ばみ・曇りが酷くないか(光量不足の主因)
- [ ] ウインカー:前後左右+ハザード、点滅速度が異常に速くないか(球切れのサイン)
- [ ] ブレーキランプ:全灯点灯(ハイマウント含む)— 誰かに踏んでもらって確認
- [ ] テールランプ・ナンバー灯・バックランプ・前後フォグの動作
- [ ] レンズの割れ・色(規定色以外の球・フィルムはNG)

### タイヤ・ホイール

- [ ] 溝の深さ:**スリップサインが出ていないか(残り溝1.6mm以上)** — 1本でも露出していれば不合格
- [ ] ひび割れ、偏摩耗、製造年(サイドウォールの4桁刻印)
- [ ] はみ出しの有無(タイヤ・ホイールがフェンダーから突出していないか)
- [ ] ホイールナットの緩み・欠品

### ガラス・ワイパー・視界

- [ ] フロントガラスの飛び石傷・ひび(運転者の視界内の損傷は不合格リスク大)
- [ ] ワイパーの拭き取りとウォッシャー液の噴射
- [ ] フロント・運転席・助手席ガラスのフィルム:**可視光線透過率70%以上**が必要(市販のスモークは大半が不適合)
- [ ] ダッシュボード上の警告灯:**エンジン・ABS・エアバッグ(SRS)等の警告灯が点灯したままだと不合格**

### 車体・外装

- [ ] 最低地上高 9cm 以上(ローダウン車は特に確認)
- [ ] マフラー:排気漏れ、腐食穴、異常な音量、後方への突き出し
- [ ] オイル・クーラント等の漏れ(駐車場所の床のシミを確認)
- [ ] フェンダーの鋭利な損傷、バンパーのぐらつき
- [ ] 発炎筒(非常信号用具)の**有効期限**または非常信号灯の搭載
- [ ] ホーンの動作、ドアミラーの固定

### 機能まわり

- [ ] ブレーキの効きと踏みごたえ(スポンジー感、片効き、異音)
- [ ] パーキングブレーキの効き(坂道で保持できるか)
- [ ] シートベルトの巻き取り・バックルの動作(全席)
- [ ] メーターの走行距離を控えておく(検査票記入用)

### 書類

- [ ] 車検証(自動車検査証)
- [ ] 自賠責保険証明書(現在のもの。新しい期間の分は車検時に加入)
- [ ] 納税証明書(自動車税の滞納がないこと。電子確認可の地域もあり)
- [ ] 点検整備記録簿(あれば — リセールにも効きます)

## AIスキルとの組み合わせ方

このチェックリストで見つかった問題は、各スキルで深掘りできます:

| 見つかった問題 | 使うスキル | 聞き方の例 |
|---|---|---|
| 警告灯が点いている | [OBD-II Diagnostic Assistant](../../skills/obd2-diagnostic-assistant/README.ja.md) | 「車検前にP0420が出ています。最短で消すのではなく直すには?」 |
| 異音・漏れがある | [Vehicle Troubleshooting](../../skills/vehicle-troubleshooting/README.ja.md) | 「車の下にオレンジ色の液だまり。車検まで2週間、緊急度は?」 |
| 自分で直したい | [Mechanic Assistant](../../skills/mechanic-assistant/README.ja.md) | 「ヘッドライトの黄ばみ除去、光量検査に通るレベルの手順は?」 |
| 車検後の整備計画 | [Vehicle Maintenance Planner](../../skills/vehicle-maintenance-planner/README.ja.md) | 「車検は通ったが次の2年で必要になる整備を予算付きで」 |
| カスタムの適合が不安 | [Vehicle Customization Advisor](../../skills/vehicle-customization-advisor/README.ja.md) | 「この車高調とマフラーの組み合わせ、車検で見られるポイントは?」 |

**プロンプトのコツ:** 「車検」という単語と、車種・年式・走行距離・改造の有無を必ず含めてください。AIは日本の保安基準の**細かい数値を誤って記憶していることがある**ため、透過率・地上高・音量などの具体的な基準値は必ず最新の公式情報(国土交通省、軽自動車検査協会)で確認してください。

## ユーザー車検を検討している人へ

1. まずこのチェックリストを済ませ、指摘されそうな箇所を潰す
2. 予備検査場(テスター屋)で光軸・サイドスリップ・排ガスを事前調整(数千円)— 初回不合格の大半はこの3つ
3. 運輸支局の検査予約(独立行政法人自動車技術総合機構の予約システム)
4. 不合格でも当日中の再検査が可能(回数制限あり)

ユーザー車検は「検査に通す」だけで整備はされない点に注意 — 24ヶ月点検は別途実施(前検査・後整備も可)してください。
