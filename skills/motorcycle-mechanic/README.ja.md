# Motorcycle Mechanic(バイク整備アシスタント)

[English](./README.md) | **日本語**

バイクの整備・診断・修理のためのAIスキルです。チェーンとスプロケットの整備、キャブレターとFIシステム、バルブクリアランス調整、タイヤとサスペンションのセッティング、冬季保管までをカバーします。

## 説明

バイク整備アシスタントは、整備士アシスタントのアプローチを二輪に持ち込みます。二輪は自分で整備する文化が強く、同時にミスの許容幅が小さい世界です。多くのライダーが自分でやる作業(チェーン整備、オイル交換、ブレーキパッド)、腰が引けがちな中級作業(バルブクリアランス、キャブ同調、フォークオイル)、そしてバイク特有の不調の診断ロジックを扱います。スポーツバイク、クルーザー、デュアルパーパス、オフロード、スクーターに対応します。

**対象:** 自分のバイクに触るすべてのライダー — 初めてのチェーン調整からエンジンフルメンテまで

## できること

- **定期整備** — オイルとフィルター交換、チェーンの清掃・注油・張り調整、スプロケットの摩耗判定、クーラント、ブレーキフルード、エアフィルター、スパークプラグ
- **キャブ・燃料系** — キャブレターの清掃とオーバーホール、ジェッティングの考え方、多気筒車の同調、ガソリンの劣化・エタノール起因の問題、FIの基礎とスロットルボディ同調
- **バルブ調整** — スクリュー式とシム式(シムアンダーバケット)の手順の違い、調整間隔が重要な理由、測定値の解釈
- **電装** — バッテリー管理(鉛 vs リチウム)、充電系のテスト(ステーター、レギュレーター/レクチファイア)、定番の配線トラブルポイント
- **車体** — タイヤ選択と摩耗の読み方、サスセッティング(サグ出し、プリロード、減衰の基礎)、ステムベアリングとホイールベアリングの点検、ブレーキ整備
- **診断** — 始動困難/始動不能、開け始めのボコつき、オーバーヒート、オイル消費、クラッチの切れ不良/滑り、ミッションの不調、ウォブルとウィーブ
- **季節対応** — 冬季保管(燃料の安定剤、バッテリーメンテナンス)、春の再稼働チェックリスト
- **適合と安全の背景知識** — チェーン/スプロケットのサイズ選択、タイヤサイズ変更の影響、「アップグレード」がハンドリングを悪化させるケース

```text
見分け方も教えてください。
```

```text

```


```text
シムアンダーバケット式エンジンのバルブクリアランス点検の手順を
```


## 推奨ワークフロー

1. **車両を特定する。** 年式、メーカー、車種、エンジン(キャブかFIか)、走行距離、改造の有無。
2. **自分の経験レベルを伝える。** 初めての人とベテランでは、適切な手順の説明が変わります。
3. **工具と部品のリストを先にもらう。** バイクの作業は専用工具(チェーンカッター、バキュームゲージ、バイク向けレンジのシックネスゲージ)が必要なことが多いです。
4. **サービスマニュアルを開いた状態で作業する。** アシスタントには説明と段取りを、正確な数値は純正マニュアルを使ってください。
5. **感想ではなく測定値を報告する。** チェーンの張りはmmで、クリアランスはmmで、サグはmmで — 数値が助言を精密にします。
6. **走る前に安全確認をもらう。** 作業後の確認リストを求めてください — バイクではアクスルナットの緩みやブレーキレバーのスカスカは警告灯ではなく、転倒に直結します。

## 制限事項

- **バイクは安全マージンが小さい。** ブレーキ、タイヤ、アクスル、ステアリング、チェーン/スプロケットの作業は転倒リスクに直結します。トルク値は必ず純正マニュアルで確認し、自信がない安全部位の作業は点検を受けてください。
- **車種ごとの違いが極端に大きい。** 同じメーカー内でも手順は大きく異なります。必ず車種別マニュアルで確認してください。
- **乗り味は遠隔では評価できません。** ハンドリングの不調(ウォブル、ウィーブ、接地感のなさ)はプロの車体点検が必要な場合があります。
- **旧車・少数販売車**は資料が乏しいことがあります。オーナーズクラブや専門店の知見は引き続き不可欠です。
- **タイヤとブレーキの選択はライダーの技量と用途に関わります** — アシスタントは公道用途では保守的に助言します。
