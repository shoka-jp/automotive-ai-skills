# Mechanic Assistant(整備士アシスタント)

[English](./README.md) | **日本語**

プロの整備士や本格的なDIYユーザーを、修理作業の最初から最後まで支援する汎用AIスキルです。初期の症状判断から、トルク値、作業手順のウォークスルー、修理後の確認までをカバーします。

## 説明

整備士アシスタントは、工場や自宅ガレージにおける「頼れる相談相手」として機能します。慣れない作業の段取り、手順や規定値の確認、修理順序の計画、よくある失敗の回避を支援します。純正整備書、メーカーのサービスキャンペーン情報、そして現場の判断を**補完**するものであり、置き換えるものではありません。

**対象:** プロの整備士、見習い整備士、経験豊富なDIYユーザー

## できること

- 修理手順のステップバイステップ解説(タイミングベルト交換、ハブベアリング交換、ヘッドガスケット作業など)
- 車両システムの仕組みの解説(点火、燃料供給、冷却、充電、ブレーキ、空調、駆動系)
- 複数の原因が考えられる症状に対する診断戦略の提案
- 一般的なトルク値の目安と締め付け順序の提示(純正整備書での確認を前提とするリマインダー付き)
- 作業に必要な工具の提案(特殊工具と代用可能な工具を含む)
- 特定のメーカー・車種・エンジン系列における典型的な故障パターンの指摘
- 整備書の表現、配線図の記号ルール、技術通報(TSB)用語の読み解き支援
- 作業難易度・標準作業時間の目安・DIYに適した作業かどうかの判断支援
- 部品選定の助言:純正 vs 社外品、リビルト vs 新品、「ついで交換」すべき部品

## プロンプト例

```text
2012年式スバル アウトバック 2.5Lのタイミングベルトを交換します。
手順を教えてください。また、ついでに交換すべき部品も教えてください。
```

```text
アルミブロックエンジンのヘッドボルトの正しい締め付け順序と
トルク管理の考え方を教えてください。なぜ順序が重要なのかも説明してください。
```

```text
お客様の車から、エンジン回転数に比例して変化し、車速には連動しない
うなり音が出ています。まずどのシステムから点検すべきですか?
```

```text
固着したブレーキキャリパーブラケットのボルトを外したい。
穏やかな方法から強硬な方法まで、選択肢を順に教えてください。
```

```text
交換が必要かどうか、どう見分ければいいですか?
```

## 推奨ワークフロー

1. **車両を正確に伝える。** 年式、メーカー、車種、エンジン、トランスミッション、走行距離。グレードによって手順が変わる場合も多くあります。
2. **作業内容または症状を伝える。** すでに点検・交換した箇所も含めてください。
3. **工具を握る前に計画を求める。** 手順全体、必要工具、部品リストを先に出してもらいます。
4. **重要な規定値は必ず確認する。** トルク値、油脂類の容量、クリアランスは、適用前に純正整備書と照合してください。
5. **対話的に進める。** 各ステップで見つかったこと(例:「テンショナーにガタがある」)を報告し、計画を調整してもらいます。
6. **修理を確認する。** 修理後の確認チェックリスト(漏れ確認、学習リセット手順、試運転の基準)を求めてください。

## 制限事項

- **純正資料の代わりにはなりません。** トルク値、容量、手順は仕向地・生産時期・仕様変更で異なります。安全に関わる数値は必ず純正整備書で確認してください。
- **見る・聞く・測ることはできません。** 診断の質は、あなたの説明と測定値の正確さに完全に依存します。
- **知識には期限があります。** ごく最近のモデル、リコール、サービスキャンペーンは反映されていない場合があります。
- **修理結果への責任は負えません。** ブレーキ、ステアリング、サスペンション、エアバッグ(SRS)、燃料系は安全に直結します。迷ったら資格を持つ専門家に委ねてください。
- **地域差があります。** 規定値や法規は市場(日本、米国、欧州など)によって異なります。
