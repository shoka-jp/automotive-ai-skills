# Body & Paint Assistant(板金・塗装アシスタント)

[English](./README.md) | **日本語**

車のボディと塗装のためのAIスキルです。へこみ・傷の評価、修理方法の選択(デントリペア/パテ/パネル交換)、色合わせとぼかしの考え方、そして「DIYの限界はどこか」の正直なガイダンスを提供します。

## 説明

板金・塗装は「どうせ簡単だろう」が高くつく領域です。このスキルは、損傷を正確に評価し(その傷はクリア層を貫通している?そのへこみはデントリペアで直る?)、正しい修理経路を選び、塗装の色が合う/合わない理由を理解し、そして何より重要なこととして、**DIYで手を出すと「何もしない」より車の価値を下げる仕事**を見分けられるようにします。磨き・研磨系の補正(ポリッシング、ペーパー研ぎ)や、板金修理をめぐる保険の文脈もカバーします。

**対象:** DIY・板金工場・現状維持で迷っているオーナー、補正(研磨)に踏み出すディテイリング愛好家、修理品質を見極めたい中古車購入者

## できること

- **損傷評価** — 傷の深さ診断(クリア層/カラー層/プライマー/金属素地。爪テストと映り込みテスト)、へこみの分類(大きさ、位置、折れ線の有無、アルミと鋼板の違い)、塗膜が切れた場合の錆リスク、樹脂パネルと金属パネルの違い
- **修理経路の選択** — デントリペア(PDR)が適用できる条件、パテ+塗装の修理、パネル修理とパネル交換の損益分岐、部分補修(スマートリペア)が効く場合と粗が見える場合
- **塗装システム** — 現代の塗装の層構造(電着、プライマー、ベース、クリア)、カラーコードの探し方、同じコードでも色が合わない理由(退色、メタリックの配向、調色)、隣接パネルへの「ぼかし塗装」が標準である理由、3コートパール の難しさ
- **DIYに適した作業** — タッチアップペンの正しい使い方(チップ埋め、平滑化)、クリア層の傷のポリッシング、ヘッドライトの黄ばみ除去、小面積の缶スプレー技法とその正直な限界、樹脂トリムやバンパー擦り傷の補修
- **補正・ディテイリング** — ポリッシングの理論(切削と仕上げ)、機械と手磨き、ペーパー研ぎのリスクとクリア層の膜厚予算、膜厚計、洗車傷の予防
- **錆修理のグレード** — 表面処理で済む錆と切開溶接が必要な錆、塗装の膨れが見た目以上に深刻な理由、修理後の防錆(空洞ワックス)
- **品質評価** — 板金工場の仕事の見極め(パネルの隙間、ゆず肌の比較、マスキング跡、オーバースプレー、ぼかしの見え方)、工場への質問リスト、まともな見積りに含まれるべき項目
- **保険の文脈** — 保険修理と修理方法の関係(純正パネルと社外パネル、格落ち・評価損の考え方)、損傷の正しい記録方法

## 推奨ワークフロー

1. **決める前に評価する。** 損傷を正確に描写してください:大きさ、パネル上の位置、深さテストの結果、わかればパネルの材質、車の年式と価値。
2. **選択肢の全体像をもらう。** 「このまま乗る」からパネル交換まで、それぞれの現実的な費用と見た目の仕上がりを聞いてください。
3. **修理を車に合わせる。** 20年落ちの通勤車とリース車では正解が違います — どちらか伝えてください。
4. **DIYの場合:** まず「期待できる正直な仕上がり」を聞き、それから技法、材料リスト、作業を台無しにする失敗モードを聞いてください。
5. **工場依頼の場合:** 引き取り前の品質チェックリストをもらい、直射日光の下で検品してください。
6. **すべて撮影する** — 修理前後の写真は保険、売却、今後のセッションのために。

## 制限事項

- **色と仕上がりは遠隔では判断できません。** メタリックの配向、退色、ぼかしの品質は「日光と目」の判断です — スキルは見るべきポイントを教えます。
- **2液ウレタン塗料とイソシアネート硬化剤は本当に有害です** — 吹き付けには送気マスクと換気が必要です。防護なしでの2液塗料の吹き付けは指導しません。
- **構造損傷は対象外です。** 骨格、エアバッグ展開、フレーム修正機が関わる損傷は設備のある鈑金工場の領域です — 外装の助言は構造が始まるところで終わります。
- **費用は地域・工場のグレード・塗料システムで大きく変わります。** レンジは交渉のための目安であり、見積りではありません。
- **DIYの結果は他のどの領域よりも腕と環境に依存します** — ホコリ、湿度、温度は、手順書が正しくても作業を台無しにします。
