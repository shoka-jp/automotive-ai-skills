# Used Car Inspector(中古車インスペクター)

[English](./README.md) | **日本語**

中古車の購入前評価を支援するAIスキルです。点検チェックリスト、要注意サインの検出、車種別の既知の弱点、価格交渉のサポートを提供します。

## 説明

中古車インスペクターは、「他人の問題」を買ってしまう事態を防ぎます。車種に特化した点検チェックリストの生成、事故修復歴やメーター改ざんの見分け方、販売文句や売り手の対応の読み解き、車台番号(VIN/車体番号)の見方を教え、点検で見つかった事実を交渉材料または「見送り」の判断に変換します。

**対象:** 経験を問わずすべての中古車購入者。車両を誠実に売りたい売り手にも有用です。

## できること

- 車種特化の購入前点検チェックリストの生成(そのメーカー・車種・世代・エンジンの既知の弱点)
- 構造化された点検の進行:外装、パネルの隙間と塗装、ガラス、タイヤ、下回り、エンジンルーム、油脂類、内装、電装、冷間始動、試乗
- 事故修復・水没(冠水)・メーター改ざんの痕跡の見分け方の解説
- 車台番号の構造と、車両履歴レポート(修復歴チェック等)でわかること・わからないことの説明
- 試乗中の症状の解釈(片流れ、振動、異音、変速の質、排気煙の色)
- 状態・走行距離・相場を踏まえた提示価格の妥当性評価
- 点検で見つかった問題点を、現実的な修理費用見積り付きの交渉材料に変換
- 個人売買・販売店それぞれへの質問リストと、はぐらかし回答の危険サインの解説
- 車両の権利関係(所有権留保、修復歴車、抹消登録など)とその影響の解説
- プロによる購入前点検(PPI)が必須となるケースと、工場への依頼内容の助言

## プロンプト例

```text
走行9.5万kmの2014年式BMW 328i(F30)を検討中です。この世代の既知の
問題箇所と、特に確認すべきポイントを教えてください。
```

```text
中古車の現車確認用に、30分でできる点検チェックリストを作ってください。
「即見送り」になる項目を最初に確認する順序でお願いします。
```

```text
売り手が「軽い接触事故、プロが修理済み」と言っています。修理の質を
どう確認すればいいですか?何を見るべきですか?
```

```text
試乗中、冷間時だけ2速から3速への変速でためらいがあり、暖まると
スムーズになりました。2013年式日産車のCVTですが、どの程度警戒すべきですか?
```

```text
車両は概ね良好ですが、タイヤ交換とフロントブレーキが必要で、オイル漏れが
少しあります。提示価格は95万円。交渉の組み立てを手伝ってください。
```

## 推奨ワークフロー

1. **売り手に連絡する前:** 出品情報を共有し、既知の問題、適正価格帯、事前に(遠隔で)聞くべき質問を確認します。
2. **現車確認の前:** 車種特化チェックリストを作成してもらい、印刷またはスマホに保存します。
3. **点検中:** チェックリストに沿って進め、すべて記録します — 疑わしい箇所は写真を撮ります。
4. **確認後:** 結果を報告し、リスク評価を求めます:購入 / 交渉 / 先にPPI / 見送り。
5. **支払い前:** 車検証・車台番号の一致と履歴を確認します。高額車・複雑な車は必ずプロの購入前点検を受けてください。
6. **購入後:** [Vehicle Maintenance Planner](../vehicle-maintenance-planner/README.ja.md) に引き継ぎ、キャッチアップ整備計画を立てます。

## 制限事項

- **車両を見ることはできません。** あなたの観察と写真の説明がすべてです。確認しなかった箇所の問題は検出できません。
- **履歴データベースへのアクセスはありません。** 修復歴チェックや車両履歴レポートの内容を要約してもらえれば解釈は支援できますが、取得はできません。
- **相場は変動します。** 価格評価は方向性の目安です。現在の近隣の出品や相場情報と照合してください。
- **プロの購入前点検は代替不可能です。** 高額車・希少車・改造車では、リフトとボアスコープでしか見つからない問題があります。
- **法的要件は国・地域で異なります**(告知義務、名義変更、保証制度など)— 現地のルールを確認してください。
