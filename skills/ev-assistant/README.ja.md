# EV Assistant(EVアシスタント)

[English](./README.md) | **日本語**

電気自動車のオーナーシップ全般を支援するAIスキルです。充電戦略、バッテリー健全性、航続距離の管理、EV特有の整備、自宅充電の設備計画、新車・中古EVの購入ガイダンスをカバーします。

## 説明

EVアシスタントは、ガソリン車の経験では答えられない疑問に対応します:バッテリーを長持ちさせる充電方法、航続距離が大きく変動する理由、EVに実際に必要な整備(そして不要になった整備)、中古EVのバッテリー評価、充電ネットワークを踏まえた長距離ドライブの計画。バッテリーEV(BEV)を中心に、注記付きでプラグインハイブリッド(PHEV)も扱います。

**対象:** EV購入検討者、新米EVオーナー、バッテリー寿命を最適化したいベテランオーナー、ガソリン車から転向する整備士

## できること

- **充電** — AC普通充電(レベル1/2)とDC急速充電の違い、コネクタ規格(CCS、NACS、CHAdeMO/チャデモ、Type 2)、充電カーブと「最後の20%が遅い」理由、公共充電の実践知識
- **自宅充電** — 普通充電設備の設置計画:回路容量、電気工事士に相談すべき負荷計算、固定式とコンセント式、スマート充電器の機能、深夜電力プランの活用
- **バッテリー健全性** — 劣化のメカニズム(経年劣化、充放電サイクル、高温、高い充電状態での保管)、日常の充電上限戦略(NMC系は日常80%など。LFP系は考え方が異なる)、SOH(健全度)数値の読み方
- **航続距離** — 実走行での変動要因(気温、速度、エアコン、地形、積載)、冬の航続距離低下、電費運転のテクニック
- **長距離計画** — 充電スポットの選び方、余裕(バッファ)の設計、充電速度を考慮したルート戦略
- **整備** — EV特有のスケジュール:タイヤ(摩耗が早い)、ブレーキフルード、エアコンフィルター、バッテリー/モーター冷却水。不要になる整備(オイル交換、マフラー、AT整備)も明示
- **中古EV購入** — バッテリー健全性の評価方法、保証の引き継ぎ、リコール履歴、価格と劣化のトレードオフ
- **安全の理解** — 高電圧系の境界と、オレンジ色のケーブルが「プロ専用領域」である理由
- **PHEV固有** — エンジンとEV両方の整備が必要な点、エンジンオイル管理の注意、最適な使い方

## 推奨ワークフロー

1. **車種とバッテリー化学系を特定する**(わかる場合)。NMCかLFPかで充電の助言は大きく変わります。
2. **使い方を説明する。** 1日の走行距離、自宅充電の可否、気候、長距離の頻度。
3. **購入判断の場合:** 現車確認の前に、その車種専用のバッテリー・保証チェックリストを依頼してください。
4. **充電設備の場合:** 分電盤の情報と電気料金プランを揃えてから、「電気工事士への相談メモ」を作ってもらいます。
5. **航続距離・電費の問題の場合:** 条件(気温、速度、エアコン使用)を報告し、正常な変動と故障を切り分けてもらいます。
6. **高電圧系に関わる問題:** このスキルで問題を理解した上で、EV整備資格(低圧電気取扱等の教育を受けた整備士)のある工場に持ち込んでください。

## 制限事項

- **高電圧系の修理は対象外です。** バッテリーパック、インバーター、オレンジケーブル系統の作業には専門教育と機材が必要です。アシスタントは概念を説明しますが、高電圧系の修理手順は案内しません。
- **充電ネットワークは変化が速い。** 充電スポットの状況、料金、コネクタの普及(NACS移行など)は変わります。最新の充電アプリで確認してください。
- **バッテリーの助言は化学系・車種に依存します。** メーカーの充電ガイダンスと矛盾する場合はメーカー指示に従ってください。
- **電気工事は有資格者が必須です。** アシスタントの役割は、工事を依頼する際に的確な質問ができるよう準備することです。
- **ソフトウェアで挙動が変わります** — OTAアップデートで機能や充電挙動が変更されることがあります。
