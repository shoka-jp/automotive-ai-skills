# Vehicle Customization Advisor(カスタマイズアドバイザー)

[English](./README.md) | **日本語**

車のカスタマイズ・改造を計画するためのAIスキルです。パフォーマンス、足回り、ホイール&タイヤ、オーディオ、ライティング、外装を、適合性・施工順序・予算・法規適合に配慮しながら組み立てます。

## 説明

カスタマイズアドバイザーは、「パーツを買い集める」のではなく「計画的に仕上げる」ための道具です。ビルドの目的(街乗り、サーキット、オフロード、ショー、牽引)を定義し、先に入れたパーツが無駄にならない順序で改造を並べ、適合性(フィッティング、オフセット、駆動系への負荷、ECUの挙動)を確認し、保証・法規上の注意点を指摘し、補助パーツや工賃まで含めた正直な予算を維持します。

**対象:** 初めての1本目から本格的なプロジェクトカーまで、すべてのエンスージアスト

## できること

- 目的と予算から一貫したビルド計画を作成:ストリート、サーキット走行会、ドラッグ、ドリフト、オーバーランド(車中泊・林道)、ショーカー、牽引仕様
- 改造カテゴリと実際の効果の解説:
  - **パワー系** — 吸気、排気、ECUチューン、過給、対応する燃料系の強化
  - **サスペンション** — ダウンサス、車高調、スタビライザー、ブッシュ、アライメントへの影響
  - **ホイール/タイヤ** — サイズ計算、オフセットとクリアランス、ロードインデックス、コンパウンド選択
  - **ブレーキ** — パッド、ローター、ステンメッシュホース、ビッグキャリパーキット、それぞれが本当に必要になる条件
  - **駆動系** — クラッチ、LSD、ファイナルギア
  - **外装/内装** — エアロ(機能 vs 見た目)、ライティング、ラッピング、シート、オーディオ
- 正しい施工順序の提案(例:パワーアップの前にブレーキとタイヤ、吸排気の後にチューン)
- 目玉パーツが必要とする補助改造と、隠れコスト(工賃、アライメント、キャリブレーション、保険)の指摘
- 保証への影響の解説と、規制されやすい領域の注意喚起:排ガス関連部品、排気音量、灯火の色・位置、ウィンドウフィルム、車高
- 中古パーツのリスク評価と、有名ブランド vs ノーブランドのトレードオフ
- 馬力の宣伝文句やダイノ数値の現実的な検証

```text

ジムカーナ、平日は通勤です。正しい順序でビルド計画を作ってください。
```


```text
NAエンジンへのターボキット装着に実際に必要な補助パーツと、
```

## 推奨ワークフロー

1. **パーツではなく目的から伝える。** 「ジムカーナのタイムを縮めたい」の方が「車高調を買うべき?」より良い計画につながります。
2. **全体像を共有する。** 車両、現在の改造内容、予算、日常の足として使い続けるか、車検・排ガス検査などの地域要件。
3. **順序付きの計画をもらう。** 工賃と補助パーツを含むフェーズ分けの費用で依頼してください。
4. **購入前に地元の法規を確認する。** 排ガス、騒音、灯火、車高の規制は国・地域・自治体で異なります。日本では保安基準と車検適合が基準になります。
5. **各パーツの波及効果を確認してから注文する。**(保証、保険、消耗、リセール)
6. **フェーズごとに再評価する。** 変更後に走ってみると、次に欲しいものが変わることはよくあります。

## 制限事項

- **法規は地域固有で、変わります。** 規制されやすい領域は指摘できますが、適法性の保証はできません。陸運局・指定工場・検査機関に確認してください。
- **排ガス関連装置の取り外し・無効化は多くの国・地域で公道走行車には違法です。** このスキルは排ガス装置や安全装置の無効化について助言しません。
- **フィッティングの限界事例は現実に存在します。** ホイール/タイヤの計算は「入る見込み」を示すもので、サスペンションストローク、キャリパークリアランス、ボディの個体差があるため現物合わせは依然として重要です。
- **保険への影響は契約ごとに異なります。** 未申告の改造は補償対象外になり得ます — 保険会社に確認してください。
- **パワーの数値は推定です。** 実際の効果は燃料、気候、チューンの質、駆動ロスの仮定で変わります。
