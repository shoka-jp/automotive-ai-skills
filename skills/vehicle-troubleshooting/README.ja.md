# Vehicle Troubleshooting(症状別トラブルシューティング)

[English](./README.md) | **日本語**

異音・振動・におい・液漏れ・警告灯・走行時の違和感など、**症状を起点に**原因を絞り込むAIスキルです。部品を買う前に、鑑別診断の手法で可能性を狭めます。

## 説明

このスキルは、OBD-IIスキャナーでは手が届かない領域を担当します:「段差でゴトゴト音がする」「甘いにおいがする」「ブレーキで左に流れる」。症状の完全なプロファイル(いつ、どこで、どんな条件で)を集め、可能性の高い原因をランク付けし、最も速く・安く候補を絞り込める判別テストを提案する — という鑑別診断のアプローチを取ります。

**対象:** 不具合に直面しているすべてのドライバーと、構造化されたセカンドオピニオンが欲しい整備士

## できること

- 症状カテゴリ別の診断:
  - **異音** — キーキー、ゴリゴリ、ゴトゴト、カンカン、ウォーン、ヒューン、カチカチ、シュー。車速・回転数・ハンドル操作・ブレーキ・段差との相関で切り分け
  - **振動** — 発生する速度域、加速時/減速時/惰性走行時、ハンドル/シート/ペダルのどこに伝わるか
  - **におい** — オイルの焦げ、クーラント(甘いにおい)、ガソリン、硫黄、クラッチ/ブレーキの焼け、カビ臭
  - **液漏れ** — 色・粘度・車体下の位置から液体を特定
  - **警告灯** — 意味、深刻度、走行継続の可否
  - **走行性** — 始動困難、始動不能、エンスト、アイドリング不調、加速のもたつき、燃費悪化、オーバーヒート
  - **電装** — バッテリー上がり、ライトが暗い、電装品の断続的な不調、「セルが回らない」と「セルは回るが始動しない」の切り分け
- 原因を判別するための的を絞った追加質問
- その車種・走行距離での原因の確率順ランキング
- 最も多くの候補を消せる、最も安い/速いテストの提案
- 緊急度判定:今すぐ停車 / そっと工場へ / 経過観察 / 外観上の問題のみ
- 診断のために症状を安全に再現・記録する方法の説明

## プロンプト例

```text
2016年式の SUV で、40km/h 以上で「ウォーン」といううなり音がして、
右カーブで大きくなります。何が原因として考えられますか?
```

```text
一晩置くとセルは元気に回るのにエンジンがかかりません。暖まっていれば
一発始動です。2010年式VW ゴルフです。どこから調べればいいですか?
```

```text
車の下、助手席側の前寄りにオレンジピンク色の液だまりがあります。
これは何の液体で、どのくらい緊急ですか?
```

```text
100〜110km/hの間だけハンドルが震え、それ以上でもそれ以下でも
スムーズです。この速度依存のパターンから何がわかりますか?
```

```text
3日乗らないとバッテリーが上がります。バッテリーもオルタネーターも
テスト結果は良好。暗電流(待機電流)の犯人はどう探せばいいですか?
```

## 推奨ワークフロー

1. **症状の全体像を説明する。** 何を感じるか、いつ始まったか、常時か断続的か、どんな条件で出るか(速度、回転数、温度、ハンドル操作、ブレーキ、天候)。
2. **追加質問に答える。** 「クラッチを踏むと変わりますか?」のような判別質問こそが診断を絞り込みます。
3. **ランク付きの候補リストをもらう。** 確率順の原因リストと緊急度判定を求めてください。
4. **判別テストを実施する。** 提案されたチェックを行い、結果を報告します — 回答のたびに候補が減ります。
5. **部品を買う前に確定させる。** お金を使う前に「これが原因だと確定できるテストは?」と聞いてください。
6. **修理に引き継ぐ。** 原因が確定したら [Mechanic Assistant](../mechanic-assistant/README.ja.md) で修理するか、確定した診断結果を工場に持ち込みます。

## 制限事項

- **言葉による診断は確率的です。** 音や感触を正確に言葉にするのは難しく、アシスタントが示すのは可能性であって断定ではありません。
- **安全は診断より優先です。** ブレーキの異常、ステアリングの問題、ガソリン臭、オーバーヒートは、まず安全に停車してから診断してください。
- **断続的な故障**は、データロガーや専門機材(オシロスコープ、シャーシイヤー、スモークマシン)が必要な場合があります。
- **多くの環境では音声・動画を直接聞けません** — 音の性質(高さ、リズム、何と連動するか)を言葉で描写してください。
- **車種固有の症状パターン**は一般的な推論では見落とされることがあります。車種別のオーナーコミュニティやサービス通報も併用してください。
