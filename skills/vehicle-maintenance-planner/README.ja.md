# Vehicle Maintenance Planner(車両整備計画アシスタント)

[English](./README.md) | **日本語**

車種・走行距離・気候・使い方に合わせた予防整備スケジュールを作成・管理するAIスキルです。

## 説明

車両整備計画アシスタントは、「5,000kmごとにオイル交換」といった一般論を、あなたの車に合わせた個別の計画に変えます。シビアコンディション(短距離走行、渋滞、牽引、酷暑・極寒、未舗装路)を考慮し、「今すぐ必要」と「もうすぐ必要」を区別し、今後の整備費用の見通しを立て、各整備の**理由**を説明することで、優先順位を自分で判断できるようにします。

**対象:** すべての車両オーナー — 初めて車を持つ人から、複数台を管理するフリート管理者まで

## できること

- 年式・メーカー・車種・エンジン・現在の走行距離から整備スケジュール全体を生成
- シビアコンディションに応じた間隔調整:短距離走行(チョイ乗り)、ストップ&ゴー、牽引、サーキット走行、極端な気候、砂塵環境
- 各整備の内容と、先送りした場合のリスクの説明
- 期限超過項目のリスク順の優先付け(安全 → エンジン寿命 → 快適性・外観)
- 各整備の費用目安(DIYでの部品代 vs 一般的な工場料金)
- DIY向きの整備と、設備・廃棄処理・キャリブレーションの面で工場が必要な整備の区別
- 季節整備の計画:冬支度、夏前の冷却系点検、タイヤ交換(夏⇔冬)
- 整備記録が不明・放置気味だった車両の「キャッチアップ計画」の作成
- 油脂類・消耗品の管理:エンジンオイル、クーラント、ブレーキフルード、ATF/CVTフルード、デフ・トランスファーオイル、各種フィルター、ベルト、スパークプラグ、タイヤ、ワイパー、バッテリー
- 整備記録の付け方と、リセールバリューを守る記録の残し方の助言

## プロンプト例

```text
走行13万kmの2017年式マツダ CX-5を記録簿なしで購入しました。
優先順位付きのキャッチアップ整備計画を作ってください。
```

```text
寒冷地で片道6kmの通勤にしか使っていません。この使い方だと
オイル交換の間隔はどう変えるべきですか?理由も教えてください。
```

```text
2016年式トヨタ ハイラックスで、15万kmから19万kmの間に必要な整備と、
工場依頼とDIYそれぞれの費用の目安を教えてください。
```

```text
2台分の12ヶ月整備カレンダーを作ってください:2020年式ホンダ CR-V
(4万km)と2011年式トヨタ ハイエース(22万km)です。
```

```text
「無交換(生涯保証)」とされるATFは本当に無交換でいいのですか?
実際にはいつ交換すべきですか?
```

## 推奨ワークフロー

1. **車両情報を伝える。** 年式、メーカー、車種、エンジン、トランスミッション形式、現在の走行距離。
2. **使い方を説明する。** 年間走行距離、1回あたりの走行距離、気候、牽引の有無、路面環境 — これが標準/シビアコンディションの判定を決めます。
3. **わかっている履歴を共有する。** 最近の整備と、飛ばしたとわかっている項目。
4. **優先順位付きの計画を求める。** 「今すぐ / 3ヶ月以内 / 12ヶ月以内」の区分で依頼してください。
5. **項目ごとにDIYか工場かを決める。** 自分でやる可能性がある項目は、難易度と必要工具も聞いてください。
6. **整備のたびに更新する。** 走行距離と完了項目を伝えて、計画を最新に保ちます。

## 制限事項

- **メーカー指定のスケジュールが優先です。** 取扱説明書・メンテナンスノートが正であり、アシスタントの間隔は根拠ある推定です。
- **費用は目安であり見積もりではありません。** 工賃は地域・工場の種類で大きく異なります。
- **セッションをまたぐ記憶は原則ありません。** 自分の整備記録を保管し、会話に貼り付けてください。
- **保証期間中の車両は要件が厳格です。** メーカー保証中は指定整備をそのまま実施し、記録(整備記録簿・領収書)を保管してください。
- **車両を直接点検できません。** ブレーキやタイヤなどの摩耗品は実測が必要です — 計画側ができるのは「いつ点検するか」の指示までです。
