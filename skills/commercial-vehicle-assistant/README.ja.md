# Commercial Vehicle & Truck Assistant(商用車・トラックアシスタント)

[English](./README.md) | **日本語**

トラック・バン・商用車のためのAIスキルです。エアブレーキシステム、運行前点検、積載物の固縛、重量コンプライアンスの基礎、そして「稼ぐ車両」の整備の現実を扱います。

## 説明

商用車は乗用車と壊れ方もコストの意味も違います:ダウンタイムは売上の損失であり、点検は法定義務であり、エアブレーキや運行記録計のような乗用車に存在しないシステムがあります。このスキルは日々の規律(運行前点検、固縛、重量配分)、商用車特有のシステム(エアブレーキ、PTO、テールゲートリフター、トレーラー連結)、そして働く車両を働き続けさせる整備計画をカバーします。小型商用バンから大型トラックまでを対象としますが、大型車の作業は法規制が厳しく地域依存であることを前提とします。

**対象:** 個人事業主ドライバー、小規模運送事業者、作業車を持つ職人、自分が操る機械を理解したいドライバー

## できること

- **エアブレーキ** — システム構成(コンプレッサー、エアタンク、ブレーキバルブ、チャンバー、スラックアジャスター)、油圧ブレーキと故障モードが違う理由、毎日のドレン(水抜き)と水分管理、低圧警告とスプリングブレーキの挙動、調整点検の考え方(測定と修理は多くの地域で有資格整備士の領域)
- **運行前点検** — 車型に応じた体系的な外周点検ルーチンの構築、監査・検査で見られるポイント、不具合の正しい記録、運行不可(整備不良)となる状態の認識
- **固縛(積載物の固定)** — 使用荷重限界とラッシング計算の考え方、チェーン/ベルト/バーの使い分け、摩擦と歯止め、車軸への重量配分、走行中の荷崩れの兆候
- **重量・寸法の基礎** — 車両総重量(GVM)・連結総重量(GCM)の概念、軸重配分、オーバーハングの基礎、過積載が法的にだけでなく機械的に(ブレーキ、タイヤ、フレーム)問題な理由
- **商用装備** — PTOの操作と整備、テールゲートリフター(油圧の管理、バッテリー消費)、冷凍機の基礎(予冷確認、燃料)、トレーラーカプラーと電気接続、第五輪とキングピンの摩耗
- **整備の経済学** — 点検起点の整備スケジュール、タイヤプログラム(操舵軸/駆動軸/トレーラー軸のポジション管理、リグルーブ・更生タイヤの考え方)、フリートツールとしてのオイル分析、働く車両のブレーキ・クラッチ寿命
- **ディーゼル駆動系** — 商用デューティサイクルでのDPF/アドブルーは [Diesel Specialist](../diesel-specialist/README.ja.md) と連携、アイドリング時間の代償
- **コンプライアンスの基礎** — 拘束時間・運行記録(タコグラフ)の概念、法定点検制度、免許区分 — いずれも一般論として扱い、詳細は地域の法規によることを明示

## プロンプト例

```text
テールゲートリフター付き7.5トンの箱車用に、10分でできる運行前点検
ルーチンを作ってください。法令・運行不可項目が先に来る順序で。
```

```text
エアブレーキのスプリング(パーキング)ブレーキの仕組みと、低圧警告が
「次の出口ではなく今すぐ停まれ」を意味する理由を説明してください。
```

```text
平ボディに機械1,200kgを積みます。固縛の計算を教えてください:
どの強度のベルトを何本、どこに掛けますか?
```

```text
配送バンのリアタイヤが2.5万kmで摩耗するのに、フロントは6万km持ちます。
毎日ほぼ同じ積載です。何を確認すべきですか?
```

```text
市街地配送ルートに変わってから、トラックのDPF再生が頻発します。
なぜですか?選択肢は何がありますか?
```

## 推奨ワークフロー

1. **車両と仕事を説明する。** 車型、総重量クラス、架装(リフター、PTO、冷凍機)、典型的な積載物、運行パターン(市街地配送、長距離、現場作業)、地域。
2. **誰が作業するかを伝える。** ドライバーレベルの日常点検と工場整備は別のスコープです — スキルは深さを調整します。
3. **点検・ルーチンの場合:** 法的な重大度順のチェックリストを作ってもらい、自分の車両に合わせて印刷してください。
4. **積載の質問の場合:** 実際の重量と寸法を提供してください — 固縛と配分の助言は数字の正確さがすべてです。
5. **故障の場合:** 商用のダウンタイムは早めのエスカレーションを正当化します —「今直す/予定を組む/停める」を明示的に聞いてください。
6. **法令は現地で確認する。** すべての規制関連の説明はオリエンテーションであり、最新のルールは運輸当局・陸運局に確認してください。

## 制限事項

- **この領域は法規制が支配的で、地域差が非常に大きい**(重量、拘束時間、点検制度、免許、固縛基準)。スキルは概念とカテゴリを教えます — あなたの地域の現行法規が正です。
- **エアブレーキの測定・修理は多くの地域で有資格整備士の領域**であり、それが正しい姿です — このスキルはシステムを理解し、点検と伝達ができるようになるためのもので、自分でブレーキを調整するためのものではありません。
- **大型の上限に近づくほど具体性は薄くなります。** 小型・中型商用車の助言は厚く、大型連結車(フルトレーラー等)については専門教育と併用する概念的な支援と考えてください。
- **固縛の助言は概念です** — 公的な固縛基準・ガイドライン(表と図を含む)が実務文書です。日本ではトラック協会等の資料も参照してください。
- **ダウンタイムの経営判断は負えません。** 稼働車両の「修理か停止か」は、アシスタントが情報提供はできても責任は持てない経営リスクです。
